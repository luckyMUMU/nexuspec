# NexusSpec 示例代码

这个目录包含了 NexusSpec 的各种示例代码，帮助你更好地理解和使用 NexusSpec。

## 示例列表

1. **基础示例**：基本的 NexusSpec 使用方法
2. **契约管理示例**：契约生命周期管理
3. **跨服务提案示例**：完整的提案流程
4. **CI/CD 集成示例**：与 GitHub Actions 等集成
5. **端到端示例**：完整的微服务场景

---

## 如何使用示例

每个示例都有详细的 README 文件，说明如何运行和理解示例。

建议按照以下顺序学习：
1. 先看[基础示例](./basic/)
2. 然后是[契约管理示例](./contract-management/)
3. 接着是[跨服务提案示例](./proposal-workflow/)
4. 最后是[端到端示例](./e2e/)

## 前置要求

- Node.js 20.19.0+
- Git
- NexusSpec CLI 已安装
- OpenSpec CLI 已安装
