## 🚀 快速开始

1. **安装 NexusSpec**
   ```bash
   npm install -g @nexus-spec/cli
   ```

2. **阅读入门指南**
   - 查看 [入门指南](./user-guide/getting-started.md)

3. **尝试示例**
   - 运行 [基础示例](./examples/basic/README.md)

## 💡 核心概念

| 概念 | 描述 |
|------|------|
| [命名空间](./user-guide/getting-started.md#3-核心概念) | 中央 Git 仓库，管理所有服务的规范和契约 |
| [服务](./user-guide/getting-started.md#3-核心概念) | 独立的微服务仓库，包含内部规范和 NexusSpec 配置 |
| [契约](./user-guide/contract-management.md) | 服务对外公开的 API/事件定义，支持版本化 |
| [提案](./user-guide/proposals.md) | 跨服务变更的结构化流程 |
