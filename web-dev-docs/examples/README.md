# Web-Dev技能示例代码库

本目录包含了各种Web开发场景的示例代码，展示了Web-Dev技能的实际应用。每个示例都包含完整的项目结构和代码，可以直接运行。

## 示例目录结构

- **portfolio/**：个人作品集网站示例
- **todo-app/**：待办事项应用示例
- **ecommerce/**：电商网站示例
- **dashboard/**：数据可视化仪表板示例
- **social-media/**：社交媒体应用示例
- **3d-scene/**：3D场景示例

## 如何使用示例

1. 进入示例目录
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 构建生产版本：`npm run build`

## 技术栈

所有示例默认使用以下技术栈：
- **前端**：React@18 + Tailwind CSS@3 + Vite
- **构建工具**：Vite
- **样式**：Tailwind CSS

## 示例详情

### 1. 个人作品集网站 (portfolio/)

一个现代、简洁的个人作品集网站，展示个人项目和技能。

**功能**：
- 响应式设计
- 项目展示
- 技能展示
- 联系表单

### 2. 待办事项应用 (todo-app/)

一个功能完整的待办事项应用，支持添加、编辑、删除和标记完成。

**功能**：
- 添加新待办事项
- 编辑待办事项
- 删除待办事项
- 标记完成状态
- 本地存储数据

### 3. 电商网站 (ecommerce/)

一个基本的电商网站，包含产品展示、购物车和结账流程。

**功能**：
- 产品展示
- 购物车管理
- 结账流程
- 响应式设计

### 4. 数据可视化仪表板 (dashboard/)

一个数据可视化仪表板，展示各种图表和数据分析。

**功能**：
- 多种图表类型
- 数据筛选
- 响应式设计
- 实时数据更新

### 5. 社交媒体应用 (social-media/)

一个基本的社交媒体应用，包含用户 profiles、帖子和评论。

**功能**：
- 用户 profiles
- 帖子发布和浏览
- 评论功能
- 响应式设计

### 6. 3D场景 (3d-scene/)

一个使用Three.js和React Three Fiber创建的3D场景示例。

**功能**：
- 3D模型展示
- 相机控制
- 灯光效果
- 交互功能
