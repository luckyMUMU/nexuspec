# Web-Dev 技能文档

本文档提供了关于 Web-Dev 技能的详细信息，包括功能概述、使用指南、培训材料、示例代码和社区支持。

## 目录结构

- **user-guide/**: 用户指南，包括功能概述、使用方法和最佳实践
- **training/**: 培训材料，包括入门教程和进阶指南
- **examples/**: 示例代码库，包含各种web开发场景的示例
- **community/**: 社区支持机制，包括FAQ和问题解答流程

## 适用场景

Web-Dev 技能专为从0到1的绿色场域Web开发场景设计，适用于：
- 创建全新的Web项目、网站、网页、Web应用或Web游戏
- 工作区为空或不包含前端代码的情况
- 遵循完整的Web开发工作流程（PRD生成 + 开发）

## 快速开始

1. 查看 [用户指南](user-guide/) 了解基本功能和使用方法
2. 阅读 [培训材料](training/) 学习如何使用Web-Dev技能
3. 浏览 [示例代码](examples/) 了解实际应用场景
4. 参考 [社区支持](community/) 获取帮助和解答
