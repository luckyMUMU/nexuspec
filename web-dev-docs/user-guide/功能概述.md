# 功能概述

## 什么是Web-Dev技能

Web-Dev技能是一个专门用于创建生产级Web界面的工具，具有高质量的设计能力。它专为从0到1的绿色场域Web开发场景设计，帮助用户快速创建全新的Web项目、网站、网页、Web应用或Web游戏。

## 核心功能

### 1. 文档生成

- **PRD生成**：根据用户需求自动生成产品需求文档
- **技术架构文档**：生成详细的技术架构文档，包括技术栈选择、路由定义等
- **文档存储**：所有文档存储在`.trae/documents/`目录中

### 2. 开发流程

- **设计思考**：帮助用户确定明确的美学方向和设计风格
- **前端实现**：使用HTML/CSS/JS、React、Vue等技术实现功能
- **美学设计**：提供详细的前端美学指南，包括排版、色彩、动画等

### 3. 技术支持

- **框架选择**：默认推荐React，也支持其他前端框架
- **工具集成**：支持Vite、Tailwind CSS等现代前端工具
- **3D支持**：对于需要3D效果的项目，提供Three.js等库的集成

## 适用场景

**适合使用Web-Dev技能的场景：**
- 需要创建全新的Web项目
- 工作区为空或不包含前端代码
- 希望遵循完整的Web开发工作流程

**不适合使用Web-Dev技能的场景：**
- 工作区已包含现有前端项目结构
- 需要对现有代码进行bug修复或功能添加
- 增量开发已建立的代码库

## 工作流程

1. **检查现有文档**：首先检查是否存在PRD和技术架构文档
2. **创建必要文档**：如果文档缺失，生成PRD和技术架构文档
3. **开发阶段**：根据文档实现解决方案
4. **设计实现**：应用美学指南，创建视觉吸引力强的界面

## 设计原则

- **排版**：选择美观、独特的字体，避免通用字体
- **色彩与主题**：承诺一致的美学，使用CSS变量保持一致性
- **动效**：使用动画效果和微交互，提升用户体验
- **空间构图**：采用意外的布局、不对称设计、重叠元素等
- **背景与视觉细节**：创建氛围和深度，添加上下文效果和纹理

## 技术栈推荐

- **前端**：React@18 + Tailwind CSS@3 + Vite
- **初始化工具**：Vite
- **3D项目**：Three.js, @react-three/fiber, @react-three/drei

## 注意事项

- 所有生成的文档、代码注释和响应必须与用户输入的语言保持一致
- 遵循桌面优先的设计方法
- 在开始开发工作前，必须先创建或检查PRD和技术架构文档
- 文档生成后，需要用户批准才能进入开发阶段
